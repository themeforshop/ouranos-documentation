    CMS Page

    From Shopfify Admin, go to Online Store > Pages to create or config the Page.

    About us Page

    It's simple page of Shopify. You can input your Shop information into the Content field.


    Contact us Page

    It's Custom page with Submit form. View demo.

    Follow steps:

    1. Create the Page with the Template suffix is page.contact. Then input the content that you want show in the Content field.

    2. To config the Information and the Maps, you need go to Customize (Theme Editor) select view page is Contact us page or other page name that you create above.

    3. Config the options


    Wish-list Page

    It will list all wishlist products that Customer saved.

    You need create the Page with:

    • Title: must be Wish list in the first time save. If don't want this title, you can change to other Title after that. Because the fist time save, Shopify will get the page url is the title name. Must make sure that the url of this page is /pages/wish-list.

    • the Template suffix is page.wish-list.

    Compare Page

    It will list all compare products that Customer saved.

    • Title: must be Compare in the first time save. If don't want this title, you can change to other Title after that.

    • the Template suffix is page.compare.
